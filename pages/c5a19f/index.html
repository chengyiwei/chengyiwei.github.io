开发环境配置

# 下载JDK

官网下载地址 (opens new window)

# 配置环境变量

  • 变量名:JAVA_HOME 变量值:C:\Program Files\Java\jdk-20
  • 变量名:Path 变量值：%JAVA_HOME%\bin;%JAVA_HOME%\jre\bin;

# 使用vscode写JAVA

# 插件安装:

  • Language Support for Java(TM) by Red Hat (opens new window) 用于运行Java
  • Debugger for Java (opens new window) 调试Java
  • Extension Pack for Java

# 编写代码

新建一个工作区，然后打入代码

    public static void main(String[] args){
        System.out.println("hello world");
    }
}
