算法SB错误大全

# 算法SB错误大全

  • 需要 %TT\%TT%TT 时，如果答案是 F[1]+F[2]F[1]+F[2]F[1]+F[2] 那么需要再次取模
  • 动态规划转移方程有 F[i−2]F[i-2]F[i−2] 时，如果从 i=1i=1i=1 开始枚举，就需要注意数组是否会越界
  • 读入多组数据时一定要读完，否则如果读了一半中途 return 会影响到下一次的读入
  • vector<int> a(100000) a 的长度很大时，auto b=a 复制会浪费大量的时间，在不改动 b 的情况下，使用 auto& b=a 会加速
  • 在网络流，init(n) 的时候，往往不是读进来的那个 n
  • 每次 Tarjan 前，dfs 序的初始值 dfn_cnt 要赋为 0
  • 乘法逆元默认 inv[0] = 1
  • 二分答案如果太大，中间过程可能加或乘爆炸
  • 当 #deine int long long 时，int INF 需要复制 1e181e181e18
