ICPC2024网络赛第一场 游记

# ICPC2024网络赛第一场 游记

我向来不喜欢以一种悲壮的方式来学习，但真的到自己失败的时候，还是极度伤心的

# day 0

赛前 vp 了几场 icpc网络赛和 ccpc 网络赛，都处在一个恰好出线的状态，所以这次对于出线还是比较有把握的

提前一天到实验室打印板子，偶然发现，自己写的博客已经够组成一本书了，还是挺感慨的，力豪说要给我出版哈哈哈

# day1

睡到中午，吃了中饭，就去实验室准备比赛了

比赛从一点开始，到六点，开始比赛时 PTA 日常看不到题，刷了五分钟，我电脑刷出来一个 I，整个机房就我一个刷出来了，然后和队友一起看 I 好像是一个三维凸包，所以可以确定肯定不是签到

过了差不多 10 分钟，网络恢复了，开始找签到，L 是签到，力豪写了之间切了，然后发现 F 挺多人交的，我想了一下想到一个单调栈的做法然后上手写，交了一发 wa 了，后面造了一组数据 2，1，2 就把自己 hack 掉了

后面力豪想到一个笛卡尔树的做法，很快就切了，之后继续签 A，强哥读完题面和我说，我还以为是一个模拟，写了一发 WA 了，后面我读了一边题目发现强哥读漏了一个点，原来是个思维题 + if else，退了一下公式也 A 了

到此时过了 3 题，2 个小时，后面大家都在开 C 和 G，我看了一手 C，一直在想怎么组合做法，来找是否有偶数的因子，想了很久没思路就去看 G，力豪提出 n=2000n=2000n=2000 可能是 dp，并提出 b 数组可以在 O(n2log⁡n)O(n^2\log n)O(n2logn) 的复杂度下求出，于是问题转化成如何从 b 数组求 c 数组，想了很久 dp 的做法，发现走不通，我就开始思考 b 数组相关的性质，发现也走不通，突然力豪提出一个看似正确的表达式，写了一发发现不对，之后就陷入死局了，就这样一直到了考试结束，总共 3 题，最后 3 小时坐牢

发现大家打得都很难受，题目风格完全不适应，现在 ICPC 越来越偏向于 cf 风格了，之后还需要多去打打 cf，并补到 2400，然后多写写总结，像 G 这种典题做不出来是不应该的

网络赛第一场没有出线，但是我相信我们的实力出线还是没有问题的，希望第二场能正常发挥

# day2

emo了，中秋去外面散散心。
